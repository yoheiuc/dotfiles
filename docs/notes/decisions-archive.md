# 判断ログ アーカイブ

`CLAUDE.md` (L2) の「個別判断ログ」のうち、議論再開の見込みがないものをここに退避している。L2 が毎ターン読まれる context を圧縮するための保管場所であり、原本としての権威は無い（コードと L2 の現行運用が真）。

退避基準:

- `status` が `adopted` で、現在のアーキテクチャ理解に必須でない（recent / load-bearing でない）
- `status` が `rejected` で、後続の議論で覆る可能性が低い
- `status` が `superseded`（後の判断で覆された記録）

新しい退避を足すときは、L2 側の同じ行を削除してこの末尾に追記する。
